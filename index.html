Random Boolean Networks


Abstract: A random boolean network consists of N nodes, each being randomly connected to C other nodes. Each node can be in two possible states: 0 or 1. Initial states of all nodes are assigned randomly. A boolean valued function is generated. It's domain is the set of permutations of C number of 0's and 1's. It's output is either 0 or 1. Each element in the domain is randomly assigned to output either 0 or 1. The future state of each node is determined by the output of this boolean valued function operating on the states of it's connected nodes.

Usage: To study the effects of small perturbations on the stability of the network, comparing it to the effects of disturbances on existing fragile systems like genetic and neural circuits, ecosystems, atmosphere, etc. Also functions as an example of chaotic behaviour. Serves as a general modes for many automatons.

Implemented in: C/C++

Download: RBN.exe

Conway's Game of Life


Abstract: Conway's Game of Life is a zero-player game first devised by British Mathematician John Conway. It revolves around a world consisting of living and non-living cells. The future state of a cell is determined by a simple set of rules.

Usage: People have managed to construct a wide variety phenomena, ranging from trains and guns, to self-replicating entities and Turing-complete computers. It is also possible to implement Conway's game within Conway's world, which is the equivalent of making a redstone computerr in minecraft and playing minecraft on the said computer.

Implemented in: C/C++, JavaScript (online version)
